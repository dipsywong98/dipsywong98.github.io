Judge 2

Corner Detection Challenge second generation judge Authors: Dipsy Wong, Peter Tse Repo: https://github.com/hkust-smartcar/judge

Features

  1. Submit code to run in sandboxes
  2. Setting up test cases
  3. Realtime leaderboard in overall and per question
  4. Realtime grading of submission
  5. Check submission records
  6. Support different types of programming languages and grading strategies
  7. Support corner detection challenge!

Stack

Nodejs express + socketio, MongoDB + mongoose, redis + beequeue job schedualing, firejail sanbox, passportjs + gitlab openid authentication, bootstrap material, imba front end framework (I won't want to use imba again), some mocha test
