# Reimbursement System

> Automate the reimbursement process of robotics team
> Authors: [Daniel Cheung](//github.com/danvim), [Dipsy Wong](//github.com/dipsywong98)
> 

## Features

1. View team budget usage over each academic year
2. Submit claim
3. Triage claim
4. Export reimbursement form
5. Manage users, currencies

## Stack
LAMP, Laravel, Vue, SCSS
