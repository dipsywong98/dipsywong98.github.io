# Aseprite 象素畫教學

如果你是跟我一樣的手殘黨，但又需要遊戲美術，可以考慮看象素畫

## 下載Aseprite

[Aseprite](https://www.aseprite.org/)是一款專畫象素畫的開源軟件，直接下載executable要$$，所以窮人可以學我這樣自行下載源碼去自己compile executable，而官網也有完整英教學

https://github.com/aseprite/aseprite/blob/master/INSTALL.md

下面win10 安裝aseprite 流程

1. [Visual Studio Community 2019 + Desktop development + Windows 10.0.18362.0 SDK](https://visualstudio.microsoft.com/downloads/) ![](https://i.imgur.com/lnW0x5L.png)

2. [CMake (3.14 or greater)](https://cmake.org/)

3. [Ninja build system](https://ninja-build.org/)

4. [Skia-m81](https://github.com/aseprite/skia/releases) 放到`C:\deps\skia`

5. [下載aseprite source code](https://github.com/aseprite/aseprite/releases) 到`C:\aseprite`

6. 開啟cmd，跑

```bat=
call "C:\Program Files (x86)\Microsoft Visual Studio\2019\Community\Common7\Tools\VsDevCmd.bat" -arch=x64
mkdir C:\aseprite\build
cd C:\aseprite\build
cmake -DCMAKE_BUILD_TYPE=RelWithDebInfo -DLAF_BACKEND=skia -DSKIA_DIR=C:\deps\skia -DSKIA_LIBRARY_DIR=C:\deps\skia\out\Release-x64 -DSKIA_LIBRARY=C:\deps\skia\out\Release-x64\skia.lib -G Ninja ..
ninja aseprite
```

7. `C:\aseprite\build\bin\aseprite.exe` 就是executable，打開它就是了

![](https://i.imgur.com/oKwNUHx.png)

## 操作

就Ctrl+N開一個新的專案，滾輪縮放，中鍵平移，左鍵筆，右鍵刷子，當小畫家用就好
