# set up peerjs server with coturn



```bash
docker run -d --network=host unless-stopped instrumentisto/coturn \
            --user=username:password --lt-cred-mech --fingerprint
docker run -p 9000:9000 -d peerjs/peerjs-server --path=/ -p=9000
```
