Aseprite 象素畫教學

如果你是跟我一樣的手殘黨，但又需要遊戲美術，可以考慮看象素畫

下載Aseprite

Aseprite是一款專畫象素畫的開源軟件，直接下載executable要$$，所以窮人可以學我這樣自行下載源碼去自己compile executable，而官網也有完整英教學

https://github.com/aseprite/aseprite/blob/master/INSTALL.md

下面win10 安裝aseprite 流程

  1. Visual Studio Community 2019 + Desktop development + Windows 10.0.18362.0 SDK

  2. CMake (3.14 or greater)

  3. Ninja build system

  4. Skia-m81 放到C:\deps\skia

  5. 下載aseprite source code 到C:\aseprite

  6. 開啟cmd，跑

  1. C:\aseprite\build\bin\aseprite.exe 就是executable，打開它就是了

操作

就Ctrl+N開一個新的專案，滾輪縮放，中鍵平移，左鍵筆，右鍵刷子，當小畫家用就好

TiledMap

示範怎弄16x16 的tiledmap，先開一個160x160和一個16x16的sprite。我們會在16x16畫，再複製貼上到160x160的大spritesheet上面。

  1. 啟動grid settings

  2. 到16x16開啟tile mode，然後便會很有趣地自己重複，就不用人手控制tile之間的縫合

按Ctrl可以看到原本大小的方框

有個做pixel art 的小提示，即使做的過程覺得很醜，只要縮小看順眼就可以了，因為沒有人會放那麼大來玩

  1. 重選複製貼上到160x160那邊 我們大槪要做這些種類的地形

先做面朝上的牆吧

  1. 先設tile in X Axis ，這樣就不會在Y Axis 重複

  1. 畫下面

  2. 朝左跟朝右的也是差不多

  3. 角落就是複製兩條邊再重製那個角，例如左上角就是左邊跟上邊

  1. 複製左邊和右邊，合在一起就是左右邊

  1. 上下邊同理

  1. 柱子就把左右邊的上邊跟下邊並在一起

  1. export!
